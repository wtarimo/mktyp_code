How to Start

From now on we will be writing our code inside a nodejs/express webserver. Our learning goals for this week are to understand the core structure of an HTML page that has been styled using Twitter Bootstrap

We will cover the use of row and column classes to layout a page

We will cover the use of p,h1,h2,h3,... tags inside a div

We will cover links, images, video, and audio inside a styled page

Next Steps

Our next step will be to create interactive websites that use Javascript to read what a customer types in, do some computation, and write out the answer! This will give us an introduction to Javascript and how to use it in a webpage.

We will then explore single page webapps where the entire website is in a single page and javascript is used to make sure that at most one div is visible at a time. These Single Page Apps (SPAs) are useful when creating mobile apps as the entire app can reside on the mobile as a single page.

The next step will be put the SPA onto a mobile app using Cordova/Phonegap and the Cordova App.

After than we will look into the use of AJAX to read and store data in the cloud!
